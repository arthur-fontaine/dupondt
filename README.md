### Prerequisites

- [Node.js and npm](https://nodejs.org/en/download/) installed on your computer.

### Getting Started

1. Clone the repository:

   ```sh
   git clone https://github.com/arthur-fontaine/dupondt.git
   ```

2. Install dependancies:

   ```sh
   cd dupondt && npm install
   ```

3. Run a simulation:

   ```sh
   npm run simulate
   ```
